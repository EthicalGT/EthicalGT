Explore My Work...

Explore a showcase of innovative projects crafted with passion, purpose, and a drive to make ideas come alive

EthicalDood

EthicalDood is a source code repository and educational web platform designed to help users learn programming languages through real-world web-based applications. It offers solved coding problems, question-answers, and solutions in PDF format—making it a practical resource for students and self-learners.

EthicalDood 2.0

EthicalDood 2.0 is an enhanced PHP-based web application that serves as a comprehensive source code repository platform. Building on the foundation of EthicalDood 1.0, this new version introduces powerful features like interactive forums, MCQ tests, a user upload section, and more—creating an all-in-one hub for developers to share, learn, and collaborate effectively.

EthicalDood 3.0

EthicalDood 3.0 is a Java Swing and AWT-based desktop application that brings all the powerful features of its previous web versions into an offline environment. Upgraded from 2.0, it includes everything from code repositories, forums, and MCQ tests to exciting new additions like a plagiarism detector, content management system, online compiler, video lectures, and many more.

EthicalPay

EthicalPay is a dummy and test-mode payment gateway designed specifically for students and college-level projects. It provides an easy-to-integrate solution for handling transactions in a controlled environment, allowing developers to test payment flows without real financial transactions.

LocalPal

LocalPal is a static web application built with Django and a MySQL database, designed to be a comprehensive guide for users to explore nearby attractions and social places. The application provides detailed information about various locations, helping users find interesting places to visit. It includes an admin panel for content management, where the admin can add, update, or delete information using CRUD operations, as well as gather feedback from users.

PsychBuddy

PsychBuddy is a robust Django-based web application designed to provide comprehensive mental health support. It empowers users to access a variety of therapeutic resources, book appointments, connect with therapists, and track their emotional well-being through advanced features like AI support and emotion detection. Whether you're seeking therapy, exploring wellness techniques, or engaging with a supportive community, PsychBuddy is your trusted companion on the journey to mental well-being.

Shop-G

Shop-G is a PHP-based web application designed for a seamless grocery shopping experience. The platform includes all the essential features of a modern shopping application, including cart management, product listings, user feedback, and an admin panel to manage the products. The application is powered by a MySQL database, ensuring smooth storage and retrieval of product and user information.

CineVibe

CineVibe is an open-source web application designed for movie lovers to watch, download, and book tickets for movies online. It provides a simple yet powerful platform where users can explore movie descriptions, ratings, and more. The site is managed by an admin who has full control over site content through CRUD (Create, Read, Update, Delete) operations.

Kohinoor

Kohinoor is a PHP-based web application designed for a seamless jewellery shopping experience. It provides an elegant and user-friendly interface with features such as cart management using PHP sessions, purchase handling, and admin CRUD operations for efficient management. The platform also includes a feedback system to enhance customer engagement. MySQL is used as the database for managing user data, product listings, orders, and feedback.

Kavach

Kavach is a mobile application designed for women's safety, offering a range of ultimate security features to provide immediate help in emergencies. With functionalities like SOS alerts, live location sharing, emergency calling services, and admin notifications, Kavach ensures the safety and well-being of its users in critical situations.

BabyCare

BabyCare is a Java-based desktop application designed to provide valuable information and product recommendations for newborn babies. This offline, easy-to-use application allows administrators to manage content efficiently while offering parents guidance on baby care, baby products, and baby name ideas based on different religions.

NewsHub

NewsHub is a dynamic news platform where users can view the latest news, send feedback, and interact with published articles. Admins have full control over managing news, ensuring the content stays fresh and relevant.
